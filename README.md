

# 🌀 Naruto Rasengan Simulator

Bring the power of chakra to your browser! This interactive web application uses computer vision to detect your hand movements and generate a realistic **Rasengan** energy sphere in real-time. Rotate your hand to charge it up, or use both hands to create the ultimate **Giant Rasengan**!

Built with **HTML5 Canvas**, **JavaScript**, and **MediaPipe Hands**.

<img src="/two-hand.png" width="400"> <img src="/one-hand.png" width="400">

## ✨ Features

- **🌀 Real-Time Chakra Detection**: Tracks your palm position instantly using AI.
- **⚡ Dynamic Charging**: Rotate your hand to spin and enlarge the Rasengan. The faster you rotate, the bigger it gets!
- **👐 Giant Rasengan Mode**: Detects two hands simultaneously to merge chakra into a massive energy sphere between them.
- **✨ Particle System**: Hundreds of swirling blue particles create a realistic fluid energy effect.
- **💥 Shockwave Effects**: Visual rings expand when the Rasengan reaches critical size.
- **🎨 Naruto-Style Visuals**: Authentic blue chakra colors (#00BFFF), glowing cores, and spiral patterns.
- **📱 Full Screen Immersion**: Minimalist UI with a dark, focused background for maximum impact.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **AI/ML**: [MediaPipe Hands](https://google.github.io/mediapipe/solutions/hands.html) by Google
- **Graphics**: HTML5 Canvas API (for high-performance particle rendering)
- **Camera**: WebRTC / MediaDevices API
